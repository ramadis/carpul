# Parcel-React-Boilerplate

Build React.js project using Parcel, also includes:

- reboot.css
- styled-components
- react-router 4.2
- axios

# Usage

```
git clone https://github.com/J-F-Liu/parcel-react-boilerplate.git your-project
cd your-project
yarn
yarn start
yarn build
```
